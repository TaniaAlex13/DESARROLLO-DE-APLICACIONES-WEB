Inicio

Bienvenido a nuestro proyecto de Desarrollo de Aplicaciones Web. Este sitio ha sido creado con HTML5 para demostrar el uso de etiquetas semánticas y la organización adecuada del contenido.

Quiénes Somos

Somos un proyecto académico orientado al desarrollo de aplicaciones web modernas utilizando tecnologías fundamentales como HTML5. Nuestro objetivo es aprender y aplicar buenas prácticas en la creación de sitios web organizados, accesibles y funcionales. Este proyecto busca demostrar el uso correcto de las etiquetas semánticas para estructurar el contenido de una página, facilitando su comprensión tanto para los usuarios como para los motores de búsqueda. Además, permite integrar elementos multimedia, enlaces de navegación e información de contacto. Mediante este trabajo se fortalecen las habilidades relacionadas con el diseño y desarrollo web, fomentando la creatividad, la organización de la información y la construcción de interfaces que respondan a las necesidades de los usuarios en diferentes contextos digitales.
